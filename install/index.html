How to Download & Install the Ambulance App on iPhone

Watch this quick guide to learn how to download and install the Ambulance App configuration profile on your iPhone.

✅ Step-by-Step Instructions

1. Tap “Download Ambulance App” button above.

2. Allow the download

When you see “ios.niwashibase.com is trying to download a configuration profile…”, tap Allow.

3. Profile Downloaded

Safari shows “Profile Downloaded”. Tap Close.

4. Open Settings & Go to General

Open the Settings app. In Settings, tap General.

5. VPN & Device Management

Scroll down and tap VPN & Device Management.

6. Downloaded Profile

Under Downloaded Profile, tap Ambulance App. If it’s missing, return here and download again.

7. Install

Tap Install at the top right.

8. Enter Passcode

Enter your iPhone passcode when prompted.

9. Confirm Install

On the “Warning” screen, tap Install at the top right.

10. Confirm Install Again

On the “Install Profile” screen, tap Install.

11. Profile Installed

You’ll see Profile Installed. Tap Done. The app icon will now appear in your App Library / Home Screen.
