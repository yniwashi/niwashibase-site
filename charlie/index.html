What Charlie Can Help With

Charlie answers app-support questions in English and Arabic using approved Ambulance App support knowledge.

Install and Update

Android APK install/update help, iOS profile install guidance, browser notes, and common install problems.

Access Questions

Ambulance Staff vs Other User access, Android registration basics, and visible access states such as Locked, Revoked, and Verification Required.

Find App Features

Where to find Search, Guidelines, CPR, Pediatrics, Scores, Overtime, Websites, Report Issue, What's New, and other supported app tools.

Where Charlie Works

Charlie is available in the approved Ambulance Discord and Telegram spaces.

  • Discord: use the Charlie help channel, slash commands, mentions, or replies to Charlie.
  • Telegram: use bot commands, mention Charlie, or reply to Charlie in the approved group.
Discord Telegram

Support Boundaries

Charlie is built for app support only.

  • Charlie cannot provide clinical advice, treatment guidance, medication dosing, ECG interpretation, or patient-specific answers.
  • Charlie cannot access private registration records, look up accounts, unlock access, approve users, or perform admin actions.
  • If Charlie does not know an answer from approved support knowledge, he should direct you to support instead of guessing.
