Download Android App

Setup steps

  1. Open the download link on your Android phone.
  2. Do not install the app using Google Chrome because it will block APK installation. Use Samsung Internet or Firefox.
  3. When the download finishes, open the APK from a file manager such as the My Files app.
  4. If prompted, allow installation for that app in Device Settings. Android usually opens the correct settings page automatically.
  5. Tap Install and wait for the app to finish installing.

Browser note: the download is an APK file. Some browsers block APK downloads by default.
