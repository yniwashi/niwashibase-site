Download APK Details & release notes

How to install

  1. Tap Download APK above — use Samsung Internet, Firefox, or any browser except Chrome.
  2. When the download completes, open it from your notifications or Downloads folder.
  3. If prompted, tap Allow from this source to enable sideloading.
  4. Follow the installer and tap Install.
⚠ Do not use Google Chrome to download — it may block the APK. Copy and paste the APK link into Samsung Internet or Firefox instead.

iOS Shortcuts

v1.0 — Latest Released Oct 2, 2025 iOS 15+
Install Shortcuts Details & guide

How to install

  1. Make sure the Shortcuts app is installed on your iPhone (iOS 15 or later).
  2. Tap Install Shortcuts above — this opens the RoutineHub install page.
  3. Tap Get Shortcut on RoutineHub, then Add Shortcut in the Shortcuts app.
  4. Run the shortcut from the Shortcuts app or add it to your Home Screen.

The iOS version is a collection of Siri Shortcuts, not a native App Store app. It does not require an Apple account to install from RoutineHub.
